# Website's address

https://daei-fastfood-ali.ir

# Intro
وبسایت منوی دیجیتال رستوران دایی علی کرمان، شهرک باهنر. اولین کار فرانت اند real world من. 

# Description
پروژه mobile responsive برای منوی دیجیتال و اسکن شده توسط QR code. وبسایت استاتیک و single page application به دلیل اندازه و سایز وبسایت و نیاز اساسی به بالاترین حد performance برای جلوگیری از اتلاف وقت کاربر. 

# Technologies
وبسایت از ساختار ساده SPA استفاده میکنه. یک index.html به عنوان شل برای صفحات HTML که نقش کامپوننت ها را دارد که به صورت دینامیک توسط جاوا اسکریپت لود میشن. 
عکس های header به فرمت webp تبدیل شده و در سرور هاست cache میشن.
