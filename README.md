وبسایت منوی دیجیتال رستوران دایی علی کرمان، شهرک باهنر. اولین کار فرانت اند من علاوه بر طراحی رابط کاربری و دیپلوی روی هاست. 


# Description
en: A mobile responsive digital menu. The use of signle page application(SPA) implementation due to the necessity of minimum load speed and maximal performance without the website having to refresh each time a link is clicked and the customer to be able to navigate the online menu as easily as they would with paper menus.
Besides, SPA is globally an efficient and optimal selection for mobile web apps. 

موبایل ریسپانسیو برای منوی دیجیتال. اننخاب ساختار برنامه تک صفحه ای به دلیل عدم نیاز به رفرش هر بار سایت برای هر قسمت و نیاز مشتری به دسترسی ساده و سریع به تمام منو. ساختار مناسب برای سایت های متمرکز بر موبایل ریسپانسیو بودن. ساختار تک صفحه ای عموما ساختار و انتخابی بهینه برای وب اپلیکیشن ها موبابل است. 

# How it works
en: index.html acts as a shell for all other HTML pages (components) of the SPA, and JS using navbar list links dynamically loads those html pages and their header images repectively. Header's images have been converted to .webp format and are cached after one visit. 

فایل index.html در نقش یک شل  برای صفحات HTML دیگر (کامپوننت ها) است که جاوا اسکریپت از ایتم های navbar به عنوان لینک برای اونها به صورت دینامیک محتوا لود میکند از جمله عکس های هدر.
عکس های header به فرمت webp تبدیل شده و cache میشن.
