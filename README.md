# Intro
وبسایت منوی دیجیتال رستوران دایی علی کرمان، شهرک باهنر. اولین کار فرانت اند من علاوه بر طراحی رابط کاربری و دیپلوی روی هاست. 

# Description
پروژه موبایل ریسپانسیو برای منوی دیجیتال و اسکن شده کیو آر کد. اننخاب ساختار برنامه تک صفحه ای به دلیل عدم نیاز به رفرش هر بار سایت برای هر قسمت و نیاز مشتری به دسترسی ساده و سریع به تمام منو. ساختار مناسب برای سایت های متمرکز بر موبایل ریسپانسیو بودن. 

# Technologies
وبسایت از ساختار برنامه تک صفحه ای استفاده میکند. index.html به عنوان شل برای صفحات HTML دیگر (کامپوننت ها) هست که جاوا اسکریپت از ایتم های navbar به عنوان لینک برای اونها به صورت دینامیک محتوا لود میکند از جمله عکس های هدر و صفحات HTML.
عکس های header به فرمت webp تبدیل شدهو cache میشن.
